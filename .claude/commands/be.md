调用后端开发智能体进行API设计、数据库优化和服务器端逻辑开发。

# 后端开发

后端开发智能体负责：
- API设计和开发（RESTful、GraphQL）
- 数据库设计和优化（SQL、NoSQL）
- 服务器端编程（Node.js、Python、Java）
- 认证和授权系统
- 微服务架构
- 性能优化和缓存
- 安全最佳实践
- 云服务和部署

## 使用示例

```bash
/be "实现JWT认证API"
/be "设计用户管理系统的数据库架构"
/be "优化API响应性能"
/be "实施OAuth2第三方登录"
```

等价于：`/agent backend_dev [任务描述]`
