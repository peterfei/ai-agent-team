调用前端开发智能体进行UI实现、组件开发和用户体验优化。

# 前端开发

前端开发智能体负责：
- 现代JavaScript框架（React、Vue、Angular）
- HTML5、CSS3和响应式设计
- 前端构建工具和打包器
- UI/UX实现和优化
- 跨浏览器兼容性
- 性能优化
- TypeScript和现代ES6+特性

## 使用示例

```bash
/fe "创建带有邮箱验证和记住我功能的登录表单组件"
/fe "根据设计模型实现购物车UI"
/fe "优化React应用的性能"
```

等价于：`/agent frontend_dev [任务描述]`
