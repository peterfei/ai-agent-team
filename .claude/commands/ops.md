调用DevOps工程师智能体进行部署、基础设施和CI/CD流水线管理。

# DevOps工程师

DevOps工程师智能体负责：
- CI/CD流水线设计和实施
- 云基础设施和自动化
- 容器编排和微服务
- 监控、日志和可观察性
- 基础设施即代码（IaC）
- 安全和合规自动化
- 性能优化和扩展
- 灾难恢复和业务连续性

## 使用示例

```bash
/ops "部署认证服务到生产环境"
/ops "设置Kubernetes集群"
/ops "配置CI/CD流水线"
/ops "实施监控和告警系统"
/ops "创建Docker容器化部署"
```

等价于：`/agent devops_engineer [任务描述]`
