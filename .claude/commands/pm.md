调用产品经理智能体进行产品规划、需求分析和路线图制定。

# 产品经理

产品经理智能体负责：
- 产品策略和路线图规划
- 用户研究和需求收集
- 竞品分析和市场调研
- 功能优先级排序和范围定义
- 敏捷开发方法论
- 利益相关者沟通和管理

## 使用示例

```bash
/pm "为我们的SaaS平台设计用户认证系统"
/pm "分析实时协作功能的可行性"
/pm "制定下个季度的产品路线图"
```

等价于：`/agent product_manager [任务描述]`
