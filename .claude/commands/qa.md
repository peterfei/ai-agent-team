调用QA工程师智能体进行测试、质量保证和缺陷报告。

# QA工程师

QA工程师智能体负责：
- 测试规划和策略制定
- 自动化测试框架和工具
- 手动测试方法学
- 性能和负载测试
- 安全测试和漏洞评估
- CI/CD集成和质量门禁
- 缺陷报告和跟踪
- 测试文档和报告

## 使用示例

```bash
/qa "为用户认证功能创建测试计划"
/qa "为登录API创建自动化测试"
/qa "对认证API进行负载测试"
/qa "执行安全漏洞评估"
```

等价于：`/agent qa_engineer [任务描述]`
