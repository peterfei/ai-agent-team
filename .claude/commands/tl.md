调用技术负责人智能体进行技术决策和团队协调。

# 技术负责人

技术负责人智能体负责：
- 技术决策和架构设计
- 团队协调和沟通
- 代码审查和质量控制
- 技术路线图制定
- 问题解决和风险评估
- 团队技术能力建设

## 使用示例

```bash
/tl "评估技术架构方案"
/tl "协调前后端集成问题"
/tl "制定技术规范和最佳实践"
/tl "解决项目中的技术难题"
```

等价于：`/agent tech-leader [任务描述]`
