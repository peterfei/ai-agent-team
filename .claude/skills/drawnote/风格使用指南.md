# drawnote-skill 风格使用指南

## 📚 概述

drawnote-skill 提供了多种信息图风格，让您可以根据不同的内容类型和使用场景选择最合适的视觉呈现方式。本指南将帮助您了解各种风格的特点和使用方法。

---

## 🎨 可用风格

### 1. 专业商务风格 (默认)

**适用场景**：
- 商业报告和数据分析
- 公司内部文档
- 专业演示材料
- 市场分析报告

**视觉特点**：
- 简洁大方的设计
- 清晰的信息层次
- 低饱和度配色
- 专业可靠的视觉效果

**配色方案**：
- 主色：#2C3E50（深蓝灰）
- 辅色：#3498DB（亮蓝）
- 强调色：#E74C3C（红色）
- 背景：#ECF0F1（浅灰）

**使用示例**：
```
请生成一份关于2024年Q4销售数据分析的信息图，使用专业商务风格
```

**颜色语义**：

**彩色笔使用**：

## 🚀 使用方法

### 方式1：直接指定风格名称

最简单直接的方式，明确告诉AI使用哪种风格：

```
请使用【彩色手写笔记风格】生成关于机器学习的信息图
```

```
请用【专业商务风格】创建2024年度报告信息图
```

### 方式2：描述风格特征

如果不确定风格名称，可以描述想要的视觉特征：

```
请生成一个学习笔记风格的信息图，需要：
- 多种颜色的笔标注
- 荧光笔高亮重点
- 手写字体效果
- 便签纸元素
```

```
请生成一个专业商务风格的信息图，需要：
- 简洁大方
- 低饱和度配色
- 清晰的数据展示
```

### 方式3：引用风格模板

对于已保存的风格模板，可以直接引用：

```
请参考 styles/彩色手写笔记风格.md 模板，
生成关于Python编程的学习笔记信息图
```

### 方式4：组合使用

可以在指定风格的基础上，添加特殊要求：

```
请使用彩色手写笔记风格生成Vue.js框架学习笔记，
重点用红色笔标注核心API，用黄色荧光笔高亮重要概念
```

## 🔧 自定义风格

如果现有风格不能完全满足需求，可以创建自定义风格：

### 步骤1：创建风格模板

在 `styles/` 目录下创建新的风格模板文件，例如：

```
styles/我的自定义风格.md
```

### 步骤2：定义风格要素

在模板文件中定义：
- 配色方案
- 字体系列
- 布局结构
- 特殊效果
- CSS样式代码

可以参考 `styles/彩色手写笔记风格.md` 的结构。

### 步骤3：使用自定义风格

```
请参考 styles/我的自定义风格.md 生成信息图
```

## 💡 常见问题

### Q: 默认使用哪种风格？
A: 如果不指定风格，将使用专业商务风格（原有的默认风格）。

### Q: 可以混合使用多种风格吗？
A: 可以，但建议在不同部分使用，保持每个部分风格统一。

### Q: 彩色手写笔记风格适合打印吗？
A: 非常适合！彩色打印效果最佳，黑白打印也能保持良好的层次感。

### Q: 如何让生成的信息图更符合我的需求？
A: 提供详细的内容描述，明确指定风格，并说明重点内容和期望的视觉效果。

### Q: 可以调整风格的配色吗？
A: 可以！在使用风格时提出配色要求，例如："使用彩色手写笔记风格，但主色调改为紫色系"。
